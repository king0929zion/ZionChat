## v0.31.2 更新内容

### 设置页 UI 精细打磨

- **个性化图标更新**：将原有装饰性画笔图标替换为简洁的笑脸 SVG（圆环 + 微笑弧线）
- **背景色更白**：主设置页背景从 `#FCFCFC` 升级为纯白 `#FFFFFF`；所有设置子页面背景从 `#F7F7F9` 统一改为 `#FCFCFC`，视觉更清爽
- **列表项更紧凑**：每个设置条目最小高度从 64dp 降至 54dp，垂直内边距从 16dp 降至 12dp，信息密度提升
- **内凹圆角效果**：每个设置项加入 `clip(RoundedCornerShape(12dp))`，配合条目间 2dp 间距，产生精致的内凹小圆角分割线视觉
- **文字微微加粗**：设置项标签字重从 Normal 调整为 Medium，可读性略有提升
