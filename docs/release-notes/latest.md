## ZionChat v0.41.0

- 修复主对话里同一轮内容被短时间重复发给 AI 的问题：现在发送链新增了同 payload 短窗去重，同时 App automation 的自动派发也改成更稳的一次性消费，避免一条任务在重组或状态切换后被重复补发。
- 用户消息现在支持更完整的长按工具卡，提供复制、编辑、删除等操作；工具卡改成大圆角长方形样式，阴影更轻、更松，不再是旧的硬卡片感。
- 用户消息的“编辑”已经接通真实回填：长按后选择编辑，会直接把这条消息内容带回底部输入框并拉起输入焦点，方便继续修改后重发。
- 设置页里的主题外观和配色选择器一起收口成更大的圆角，阴影也同步调浅并拉开范围，整体观感更接近当前设置页主视觉。
